<h2><a href="https://www.geeksforgeeks.org/problems/reverse-an-array/0">Reverse an Array</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">You are given an array of integers&nbsp;<strong>arr</strong>&nbsp;. Your task is to&nbsp;<strong>reverse</strong>&nbsp;the given array and return the reversed array.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:<br></strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>arr = [1, 2, 3, 4]
<strong>Output: </strong>[4, 3, 2, 1]</span><br><span style="font-size: 14pt;"><strong>Explanation: </strong>The elements of the array are 1 2 3 4. After reversing the array, the first element goes to the last position,<br>the second element goes to the second last position and so on. Hence, the answer is 4 3 2 1.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input</strong>: arr = [1]
<strong>Output: </strong>[1]</span><br><span style="font-size: 14pt;"><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">Explanation: </strong>The array has only single element, hence the reversed array is same as the original.</span></pre>
